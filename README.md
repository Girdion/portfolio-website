# Jonathan Djoko Portfolio

Personal portfolio website showcasing my experience, projects, and journey as a software engineer.

🌐 Live Website: https://jonathandjoko.vercel.app

## About

This portfolio was built to provide a simple and accessible overview of my background, technical experience, and projects.

I'm currently working as a Backend Engineer at BNI, where I contribute to enterprise banking systems and backend development. Previously, I worked as a Frontend Engineer Intern at Asuransi Astra.

The goal of this website is to present my work in a clean, minimalist format that is easy for recruiters, engineers, and hiring managers to navigate.

## Local Development

Clone the repository:

```bash
git clone https://github.com/Girdion/portfolio-website.git
cd portfolio-website
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
